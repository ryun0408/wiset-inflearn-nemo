# Nemo Real Estate Analytics Project

강남/서초 지역의 상업용 부동산 데이터를 수집하고 분석하는 프로젝트입니다.

## 주요 기능
- **데이터 수집**: Nemo API를 통한 상가 매물 데이터 자동 수집
- **데이터 분석**: 수치형/범주형 데이터의 심층 EDA 및 TF-IDF 키워드 분석
- **대시보드**: 분석 결과를 시각화하여 단일 HTML 파일로 제공

## 프로젝트 구조
- `src/`: 데이터 수집 및 분석 소스 코드
- `docs/`: 프로젝트 관련 문서
- `index.html`: 분석 결과 대시보드 (GitHub Pages 배포용)
